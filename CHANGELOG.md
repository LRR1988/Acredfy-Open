# Changelog

## [4.0.1] - 2026-03-20
- Fixed accent characters across the entire application
- Compiled and deployed to production

## [4.0.0] - 2026-03-18
- New licensing system with online activation
- Auto-update system for application and data files
- Certificate search with debounce (150ms)
- Visual expiry indicators (green/yellow/red)
- Support for Edge, Chrome and Firefox
- InPrivate/Incognito mode with isolated user data directory
- 3-second policy loading page for reliable certificate selection
- Edge Startup Boost auto-disabled to prevent stale policies

## [3.0.0] - 2026-02-15
- Added bookmark management (accesos)
- Wildcard URL patterns for government site subdomains
- Cl@ve gateway support (pasarela.clave.gob.es)
- Certificate enrichment: DNI, type, org, expiry status

## [2.0.0] - 2026-01-10
- Multi-certificate support with search
- Registry policy writing for Chrome and Edge
- PowerShell certificate store reader

## [1.0.0] - 2025-12-01
- Initial release
- Single certificate activation
- Basic URL input and browser launch
