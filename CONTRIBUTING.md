# Contributing to Acredfy

Thank you for your interest in contributing to Acredfy!

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Commit your changes (`git commit -m 'Add my feature'`)
4. Push to the branch (`git push origin feature/my-feature`)
5. Open a Pull Request

## Reporting Issues

Please use the [GitHub Issues](https://github.com/LRR1988/Acredfy-Open/issues) page to report bugs or request features.

## Development Setup

### Requirements
- Python 3.10+
- Windows 10/11
- FNMT digital certificates installed in Windows certificate store

### Running in development
```bash
python Acredfy.pyw
```
